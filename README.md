## Description


[Shopping-api](https://github.com/li195888/Shopping-api)
安装[Mysql](https://dev.mysql.com/downloads/)数据库

## Installation

```bash

$ 全局安装

$ npm install express-generator --save -g

$ 快速搭建服务

$ express Shopping_api

$ 下载依赖包

$ npm install

```

## Running the app

```bash
# development
$ npm start

```
